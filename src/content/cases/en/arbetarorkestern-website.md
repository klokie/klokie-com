The website for Arbetarorkestern, the collective world-music ensemble I play in.
A bilingual (Swedish/English) Astro site covering gigs, news, and live video,
with content authored as plain markdown in an Obsidian vault and published to
Cloudflare on every push. Posters and photos are served as responsive images
from R2. It's the reference build for the klokie.com web stack — the same
pipeline this very site runs on.
