# Uses

What I build and make music with. A living page — inspired by
[uses.tech](https://uses.tech). _(First draft — refine the specifics.)_

## Editor & shell

- **Claude Code** — my primary way of writing and shipping code now.
- **Cursor** and **VS Code** for hands-on editing.
- **Obsidian** — my second brain. This whole site is authored as plain
  markdown in an Obsidian vault and published on every push.
- **Ghostty** terminal, **zsh** + oh-my-zsh.

## Web stack

- **Astro** for content sites, **Next.js** for interactive apps.
- **TypeScript** everywhere, **pnpm**, **Node** via **fnm**.
- **Cloudflare** — Workers/Pages, R2 for media, custom domains.
- **Supabase** and **Neon** (Postgres) for data.
- **PostHog** (analytics) and **Sentry** (errors), EU-hosted.

Twenty-six years of web work sit behind these choices — the current
favorites, not the whole story. See [things I've worked on](/work/).

## AI

- **Claude** (Anthropic) for building, writing, and reasoning.
- Self-hosted automations on a home Linux server running scheduled agents —
  morning briefings, inbox triage, and the search index behind this site.

## Hardware

- **Apple Silicon Macs** (M-series) for everything day to day.
- A **Linux home server** for always-on jobs and backups.
- iPhone + Apple Watch for mobile and the occasional prototype.

## Music

- **Bass** and **guitar** — see [things I've played on](/music/).
- EBS overdrive on the bass side; a rotating cast of amps and pedals.
